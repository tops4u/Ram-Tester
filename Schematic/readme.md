## Build

The PCB Files have been splited in a ThruHole (conventional - easy to build) and a SMD Version. Please use the different file folders for the desired version. 

Both Versions have the the possiblity to add a I2C OLED Display, but can also be used without.

Current PCB Design for SMD And TH.

If you want to test 4116 RAM with this tester, a 4116 adapter board is now available. It provides the required -5V and +12V for this RAM. See the 4116 Adapter subdirectory for details.
